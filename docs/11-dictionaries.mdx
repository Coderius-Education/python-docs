# Dictionaries

Een dictionary slaat waarden op met een **sleutel** (key) in plaats van een volgnummer. Je kunt het zien als een woordenboek: je zoekt een woord op (de sleutel) en krijgt de betekenis (de waarde).

---

## Predict

Bekijk de onderstaande code. Wat denk je dat er op het scherm verschijnt? Hoeveel regels output verwacht je?

```python
leerling = {
    "naam": "Sara",
    "leeftijd": 15,
    "klas": "3B"
}

print(leerling["naam"])
leerling["leeftijd"] = 16
leerling["hobby"] = "programmeren"

print(leerling["leeftijd"])
print(leerling["hobby"])
print(len(leerling))
```

Schrijf je voorspelling op voordat je verder leest.

## Investigate

Kopieer de volgende code naar de Playground en beantwoord de vragen:

```python
leerling = {"naam": "Sara", "leeftijd": 15, "klas": "3B"}

# Methode 1: Alleen de sleutels
for sleutel in leerling:
    print(sleutel)

print()

# Methode 2: Sleutels én waarden
for sleutel, waarde in leerling.items():
    print(f"{sleutel}: {waarde}")

print()

# Check of een sleutel bestaat
print("naam" in leerling)
print("school" in leerling)
```

1. Wat is het verschil tussen `for sleutel in leerling:` en `for sleutel, waarde in leerling.items():`?
2. Wat geeft `"school" in leerling` terug? Waarom?
3. Wat gebeurt er als je `print(leerling["school"])` probeert op te vragen terwijl die sleutel niet bestaat? (Probeer het!)
