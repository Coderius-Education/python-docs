# Jij als variabele

:::info Wat je al kent
Je kunt tekst op het scherm tonen met `print()`. In deze les voeg je daar iets nieuws aan toe: **variabelen**. Zo wordt je visitekaartje uit les 1 een stuk slimmer.
:::

In les 1 stond je naam hardcoded in de tekst:

```python
print("Naam: Alex Jansen")
```

Wat als je naam verandert — of als je het programma wil aanpassen voor iemand anders? Dan moet je elke `print()` handmatig aanpassen. Dat is omslachtig.

Met **variabelen** sla je een waarde op onder een naam. Die naam gebruik je daarna overal.

## Hoe werkt een variabele?

```python
naam = "Alex Jansen"
leeftijd = 15

print(naam)
print(leeftijd)
```

- `naam` is de **naam** van de variabele
- `"Alex Jansen"` is de **waarde**
- `=` slaat de waarde op in de variabele

Uitvoer:
```
Alex Jansen
15
```

## Datatypes

Niet alle waarden zijn hetzelfde soort. Voor nu concentreren we ons op de twee belangrijkste:

| Type | Voluit | Voorbeeld | Wanneer |
|------|--------|-----------|---------|
| `str` | String | `"Hallo"` | Tekst (altijd tussen aanhalingstekens) |
| `int` | Integer | `15` | Geheel getal |

```python
naam = "Alex"          # str
leeftijd = 15          # int
```

Je kunt het type opvragen met `type()`:

```python
print(type(naam))       # <class 'str'>
print(type(leeftijd))   # <class 'int'>
```

## Investigate

Wat gebeurt er als je de variabelenamen en waarden omdraait, zoals hieronder? Probeer het uit.

```python

```

<details>
<summary>Bekijk het antwoord</summary>

```

```

Het `=`-teken werkt maar in één richting: waarde rechts → variabelenaam links. `"Alex" = naam` is dus ongeldig.

</details>

## Variabelen veranderen

Je kunt de waarde van een variabele op elk moment aanpassen:

```python
score = 0
print("Score:", score)

score = 10
print("Score:", score)

score = score + 5
print("Score:", score)
```

Uitvoer:
```
Score: 0
Score: 10
Score: 15
```

`score = score + 5` betekent: neem de huidige waarde van `score`, tel er 5 bij op, en sla het resultaat op in `score`.

---

## Er gaat iets mis

Dit is een veelgemaakte fout:

```python
print(naam)
naam = "Alex"
```

**Foutmelding:**
```
NameError: name 'naam' is not defined
```

**Waarom?**
Je probeert `naam` te gebruiken voordat je hem hebt aangemaakt. Python voert code van boven naar beneden uit — een variabele bestaat pas ná de regel waar je hem aanmaakt.

**Oplossing:** Maak de variabele aan vóór je hem gebruikt.
