# Jouw naam op het scherm

Welkom bij je eerste les Python! Je schrijft je eerste programma: een digitaal visitekaartje.

Om iets op het scherm te tonen, gebruik je `print()`. Dat is de eerste instructie die je leert.

## Hoe werkt print()?

```python
print("Hallo, wereld!")
```

Python voert je code van **boven naar beneden** uit — elke regel is een instructie. Dit programma heeft één instructie en toont één zin.

Meerdere regels? Meerdere `print()`-aanroepen:

```python
print("Hallo!")
print("Ik leer Python.")
print("Dit is makkelijk.")
```

Uitvoer:
```
Hallo!
Ik leer Python.
Dit is makkelijk.
```

:::caution Let op
Tekst moet altijd **tussen aanhalingstekens** staan. Vergeet je ze, dan begrijpt Python je code niet.
:::

## Investigate

Wat gebeurt er als je de volgorde van de `print()`-regels omdraait? Probeer het:

```python

<details>

```

De **volgorde van je code** bepaalt de volgorde van de uitvoer.

</details>

## Er gaat iets mis

Dit is een veelgemaakte fout:

```python
print(Hallo!)
```

**Foutmelding:**
```
SyntaxError: invalid syntax
```

**Waarom?**
Tekst die je wilt printen moet tussen aanhalingstekens staan. Python ziet `Hallo!` zonder aanhalingstekens als een naam (variabele), niet als tekst — en die naam bestaat niet.

**Oplossing:** Zet aanhalingstekens om de tekst: `print("Hallo!")`.
