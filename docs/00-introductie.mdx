# Welkom bij Python!

Python is een van de populairste programmeertalen ter wereld. Het wordt gebruikt voor websites, games, data-analyse, kunstmatige intelligentie en nog veel meer. En het mooiste: Python is makkelijk om te leren!

## Wat ga je bouwen?

In elke les bouw je iets werkends. Nieuwe concepten leer je altijd samen met wat je al weet:

1. **Jouw naam op het scherm** — je eerste programma: een digitaal visitekaartje
2. **Jij als variabele** — het visitekaartje slimmer maken met variabelen
3. **De rekenmachine** — berekeningen uitvoeren met variabelen
4. **Slimme berichten** — uitvoer mooi opmaken met f-strings
5. **Ja of nee?** — beslissingen nemen met `if` en `else`
6. **Herhaling** — code automatisch herhalen met de for-loop
7. **Zolang als...** — herhalen zolang iets waar is met de while-loop
8. **Herbruikbare code** — functies schrijven zodat je code niet herhaalt
9. **Functies met waarden** — functies die iets ontvangen en teruggeven
10. **Lijsten** — meerdere waarden bijhouden
11. **Opzoeken met een sleutel** — waarden opslaan en terugvinden met dictionaries

## Hoe werkt het?

Elke les heeft:
- **Predict** — denk eerst na over wat de code doet voordat je hem uitvoert
- **Run** — voer de code uit in de Playground en vergelijk met je voorspelling
- **Investigate** — experimenteer met de code om te begrijpen hoe het werkt
- **Modify** — pas de code stap voor stap aan
- **Make** — aan het einde bouw je iets werkends

Je hoeft niks te installeren. Alle code kun je direct in de browser draaien!
