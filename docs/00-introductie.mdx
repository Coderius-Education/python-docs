# Welkom bij Python!

Python is een van de populairste programmeertalen ter wereld. Het wordt gebruikt voor websites, games, data-analyse, kunstmatige intelligentie en nog veel meer. En het mooiste: Python is makkelijk om te leren!

## Wat ga je leren?

In deze tutorial leer je stap voor stap de basis van Python:

1. **Print** — tekst op het scherm tonen
2. **Variabelen** — waarden opslaan en gebruiken
3. **Rekenen** — berekeningen maken
4. **Strings** — werken met tekst
5. **Booleans** — waar of niet waar
6. **If/else** — beslissingen nemen
7. **For-loop** — code herhalen
8. **Functies** — code hergebruiken
9. **Parameters en return** — functies slimmer maken
10. **Lijsten** — verzamelingen van waarden
11. **Dictionaries** — waarden opslaan met een sleutel

## Hoe werkt het?

Elke les heeft:
- **Uitleg** met voorbeelden
- **Oefeningen** die je direct in je browser kunt uitvoeren
- **Tips** als je vastloopt
- **Oplossingen** om te controleren

Je hoeft niks te installeren. Alle code kun je direct in de browser draaien!
