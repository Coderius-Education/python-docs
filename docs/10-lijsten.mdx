# Lijsten

Een lijst (`list`) is een variabele waarin je meerdere waarden tegelijk opslaat, in een vaste volgorde. Denk aan een boodschappenlijstje, een klassenslijst, of een reeks cijfers.

---

## Predict

Bekijk de onderstaande code. Wat denk je dat er op het scherm verschijnt? Let goed op de nummering: bij welk getal begint Python te tellen?

```python
kleuren = ["rood", "groen", "blauw", "geel"]

print(kleuren[0])
print(kleuren[2])
print(kleuren[-1])
print(len(kleuren))
```

Schrijf je voorspelling op voordat je verder leest.

## Run

Zet de code in de [Playground](/playground) en voer het uit.

<details>
<summary>👀 Kwam dit overeen met je voorspelling?</summary>

```text
rood
blauw
geel
```

Python telt altijd vanaf **0**! Dus `kleuren[0]` is "rood" (het eerste element), `kleuren[2]` is "blauw" (het derde element). Met `[-1]` pak je altijd het **laatste** element. En `len()` geeft het totaal aantal elementen.

## Investigate

Kopieer de volgende code naar de Playground en beantwoord de vragen:

```python
dieren = ["kat", "hond", "vis"]

dieren.append("vogel")
print(dieren)

dieren.remove("hond")
print(dieren)

laatste = dieren.pop()
print(f"Verwijderd: {laatste}")
print(dieren)
```

1. Wat doet `.append()`? Waar komt het nieuwe element terecht (aan het begin of het einde)?
2. Wat doet `.remove()`? Wat gebeurt er als je een element probeert te verwijderen dat niet in de lijst staat?
3. Wat doet `.pop()`? Wat is het verschil met `.remove()`?
4. Probeer ook eens: `dieren.insert(0, "hamster")`. Wat doet dit?
