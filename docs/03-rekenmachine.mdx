# De rekenmachine

:::info Wat je al kent
Je weet hoe je variabelen aanmaakt (`naam = "Alex"`, `leeftijd = 15`) en hoe je ze print. In deze les leer je iets nieuws: **rekenoperatoren** (zodat Python voor je rekent).
:::

Tot nu toe stonden alle waarden direct in je programma, maar we deden er niets mee. Aan het einde van deze les bouw je een handige rekenmachine die automatisch alle uitkomsten berekent als je de variabelen aanpast.

## Rekenoperatoren

Python kan rekenen als een calculator:

| Operator | Betekenis | Voorbeeld | Resultaat |
|----------|-----------|-----------|-----------|
| `+` | Optellen | `5 + 3` | `8` |
| `-` | Aftrekken | `10 - 4` | `6` |
| `*` | Vermenigvuldigen | `3 * 7` | `21` |
| `/` | Delen | `10 / 4` | `2.5` |
| `//` | Geheel delen | `10 // 3` | `3` |
| `%` | Rest (modulo) | `10 % 3` | `1` |
| `**` | Macht | `2 ** 3` | `8` |

```python
print(5 + 3)    # 8
print(10 // 3)  # 3  (hoe vaak past 3 in 10?)
print(10 % 3)   # 1  (wat blijft er over?)
```

## Rekenen met variabelen

Je kunt je variabelen uit les 2 direct gebruiken in berekeningen:

```python
prijs = 12.50
aantal = 4
totaal = prijs * aantal

print("Totaal:", totaal)  # 50.0
```

## Rekenvolgorde

Net als bij wiskunde: vermenigvuldigen en delen gaan vĂłĂłr optellen en aftrekken. Gebruik haakjes om de volgorde te sturen:

```python
print(2 + 3 * 4)    # 14  (niet 20!)
print((2 + 3) * 4)  # 20
```

## Investigate

Wat is het verschil tussen `10 / 2` en `10 // 2`? En wat doet `10 % 2`?

```python
print(10 / 2)
print(10 // 2)
print(10 % 2)
```

<details>
<summary>Bekijk het antwoord</summary>

```
5.0
5
0
```

- `10 / 2` geeft `5.0` â€” altijd een `float`, ook als het resultaat een geheel getal is
- `10 // 2` geeft `5` â€” altijd een `int`
- `10 % 2` geeft `0` â€” de rest is 0, dus 10 is exact deelbaar door 2

Dit is waarom `%` handig is om te controleren of een getal **even** is: als `getal % 2 == 0`, dan is het even.

</details>
