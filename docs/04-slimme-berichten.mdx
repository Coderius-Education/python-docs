## f-strings

Een f-string is een string die je variabelen direct kunt insluiten. Zet een `f` vóór de aanhalingstekens, en zet variabelen tussen `{}`:

```python
naam = "Alex"
leeftijd = 15

print(f"Hallo, ik ben {naam} en ik ben {leeftijd} jaar oud.")
```

Uitvoer:
```
Hallo, ik ben Alex en ik ben 15 jaar oud.
```

Je kunt ook **berekeningen** doen in een f-string:

```python
leeftijd = 15
print(f"Volgend jaar ben ik {leeftijd + 1} jaar.")
```

## String-methoden

Strings hebben ingebouwde functies ("methoden") die je kunt aanroepen met een punt:

```python
tekst = "hallo wereld"

print(tekst.upper())    # HALLO WERELD
print(tekst.lower())    # hallo wereld
print(tekst.capitalize())  # Hallo wereld
print(len(tekst))       # 11 (inclusief spatie)
print(tekst.replace("wereld", "python"))  # hallo python
```

---

## Investigate

Wat is het verschil tussen `.upper()`, `.lower()` en `.capitalize()`? Probeer dit uit:

```python
naam = "alex JANSEN"

print(naam.upper())
print(naam.lower())
print(naam.capitalize())
```

<details>
<summary>Bekijk het antwoord</summary>

```
ALEX JANSEN
alex de vries
Alex de vries
```

- `.upper()` maakt alles hoofdletter
- `.lower()` maakt alles kleine letter
- `.capitalize()` maakt alleen de allereerste letter een hoofdletter, de rest kleine letters

:::info Let op: niets is permanent
Deze methodes veranderen de variabele `naam` **niet** permanent. Als je `print(naam)` toevoegt aan het einde van het programma, zie je dat de originele waarde `"alex JANSEN"` nog steeds hetzelfde is. Wil je de wijziging opslaan? Overschrijf dan de variabele:
`naam = naam.upper()`
:::

</details>

## Er gaat iets mis

Dit is een veelgemaakte fout:

```python
naam = "Alex"
leeftijd = 15

print(f"Hallo {naam}, je bent {leeftijd} jaar.")
print(f"Volgend jaar ben je {leeftijd + 1 jaar.")
```

**Foutmelding:**
```
SyntaxError: f-string: expecting '}'
```

**Waarom?**
De accolade `{` is geopend maar niet gesloten. Alles tussen `{` en `}` moet geldige Python-code zijn — je kunt geen vrije tekst (zoals `jaar`) toevoegen bínnen de accolades.

**Oplossing:** Zet de tekst buiten de accolades:

```python
print(f"Volgend jaar ben je {leeftijd + 1} jaar.")
```

## Revisie: rekenmachine mooier maken

Herinner je de rekenmachine van les 3? Met f-strings maak je de uitvoer veel netter. Vervang de `print("Som:", som)`-stijl door f-strings:

```python
a = 10
b = 4

som = a + b
quotient = a / b

# Oud (les 3):
print("Som:", som)
print("Quotiënt:", quotient)

# Nieuw (met f-strings):
print(f"Som:      {som}")
print(f"Quotiënt: {quotient}")
```

## Make: Gepersonaliseerde uitnodiging

Bouw een programma dat een feestuitnodiging genereert. Het programma:

1. Een variabele heeft voor de naam van de genodigde
2. Een variabele heeft voor de naam van het evenement
3. Een variabele heeft voor de datum
4. Print een nette uitnodiging zoals hieronder

Verwachte uitvoer (met variabelen "alex", "Eindfeest" en "20 juni"):
```
╔══════════════════════════════╗
   Uitnodiging voor ALEX
╚══════════════════════════════╝

Je bent uitgenodigd voor: Eindfeest
Datum: 20 juni

We hopen je te zien!
```

<CodeExercise>{`naam = "Alex"

print()
print(f"Je bent uitgenodigd voor: ...")
print(f"Datum: ...")
print()
print("We hopen je te zien!")`}</CodeExercise>

<details>


</details>
