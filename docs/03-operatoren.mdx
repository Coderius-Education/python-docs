# Rekenen

In Python kun je de computer als rekenmachine gebruiken.

## Rekenoperatoren

| Operator | Betekenis | Voorbeeld | Resultaat |
|----------|-----------|-----------|-----------|
| `+` | Optellen | `5 + 3` | `8` |
| `-` | Aftrekken | `10 - 4` | `6` |
| `*` | Vermenigvuldigen | `3 * 7` | `21` |
| `/` | Delen | `10 / 3` | `3.333...` |
| `//` | Geheel delen | `10 // 3` | `3` |
| `%` | Rest (modulo) | `10 % 3` | `1` |
| `**` | Macht | `2 ** 3` | `8` |

```python
print(5 + 3)     # 8
print(10 - 4)    # 6
print(3 * 7)     # 21
print(10 / 3)    # 3.333...
print(10 // 3)   # 3
print(10 % 3)    # 1
print(2 ** 3)    # 8
```

## Oefening 2

Reken 135 minuten om naar uren en resterende minuten. Gebruik `//` en `%`.

<CodeExercise>{`totaal_minuten = 135

minuten = ...

print(totaal_minuten, "minuten =", uren, "uur en", minuten, "minuten")`}</CodeExercise>

<details>
<summary>ðŸ’¡ Tip</summary>

`//` geeft het hele aantal uren (`135 // 60 = 2`). `%` geeft de rest (`135 % 60 = 15`).

</details>

<details>
<summary>âœ… Oplossing</summary>

```python

```

</details>
